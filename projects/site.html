This Website

2020 - Ongoing

This website is my first foray into serious web development since freshman year of high school when I took an introductory web design class. Since January 18th, 2020, I have been working on developing this website from scratch, using skills and techniques I learned in that class five years ago, as well as new ones I've picked up along the way.

If you are interested in learning about web development and gaining a very useful skill, check out W3Schools. Their resources have been incredibly helpful as I learn more about HTML and try to incorporate more interesting things into my site.
